TapBack Command

Prototype web — iPhone 16

Clique sur Tap Back (ou double-clic sur le châssis / touche T) : la mini-fenêtre du haut s'affiche sur la page courante (accueil ou app) — l'app ne s'ouvre pas. Choisis ensuite une action.

Astuce : la barre du bas (home indicator) ou la touche H ramène à l'accueil pour rejouer le scénario « Tap Back depuis une autre app ».

⚠️ Sur un vrai iPhone, iOS n'autorise pas d'overlay par-dessus les autres apps : le Tap-Back amène TapBack Command au premier plan sur la mini-fenêtre. Cette maquette reproduit fidèlement ce comportement.
