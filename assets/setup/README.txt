Guide « Mettre en place » (Réglages de l'app)
==============================================

Dépose ici tes captures d'écran qui montrent comment configurer
le raccourci, dans l'ordre :

  etape_1.png
  etape_2.png
  etape_3.png
  ...

Formats acceptés : .png, .jpg, .jpeg, .webp
Les images sont affichées dans l'ordre alphabétique du nom de fichier
(etape_1, etape_2, …) dans la section Réglages → « Mettre en place ».

Après avoir ajouté/retiré des images, relance un build
(flutter run / nouvelle archive) pour qu'elles soient embarquées.
